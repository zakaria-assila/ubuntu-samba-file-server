# Logs & Troubleshooting

## Goal
Know where to check errors and how to debug common issues.

## What to Check
- Samba service status
- Auth failures
- Permission denied events
- Connectivity / port issues

## Common Problems
- Works in smb.conf but fails due to Linux filesystem permissions
- Wrong group membership
- Firewall blocking TCP 445
- Incorrect credentials / cached credentials on Windows
