
# Backup & Restore

## Goal
Back up critical configuration and verify restore.

## What to Back Up
- Samba config: `/etc/samba/smb.conf`
- Share folder structure (optional)
- Any custom scripts

## Restore Test
- Restore config file
- Restart samba
- Confirm Windows access still works
